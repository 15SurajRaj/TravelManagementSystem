# TravelManagementSystem

## 📌 Project Overview
The **Travel Management System** is a Java-based desktop application designed to manage hotel bookings and client records efficiently.  
It provides a simple and user-friendly interface built with **Java AWT** and stores booking details securely in an **SQL database** using **JDBC**.

---

## ✨ Features
- 🏨 Manage client data for hotel bookings  
- 🔄 Multiple user interfaces with Java AWT  
- 💾 Store and retrieve booking details from SQL database  
- 🔐 Secure data handling with JDBC connectivity  
- 📊 Easy to extend for additional features (e.g., travel packages, payment integration)

---

## 🛠️ Tech Stack
- **Programming Language:** Java  
- **GUI Framework:** Java AWT  
- **Database:** SQL (MySQL / any RDBMS)  
- **Database Connectivity:** JDBC
